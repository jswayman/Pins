RELEASE NOTES
v1.3 JULY 2026 NEW
NEW FEATURES
  • Display Name support — set a custom display name shown on leaderboards (distinct from your username)
  • Profile modal — update display name, phone, email, avatar, and password from within the app
  • Avatar picker — choose from DiceBear styles, emoji, or upload a custom photo
  • Share / Invite — use the 📲 Invite button on the leaderboard to share a pool join link via native share sheet or clipboard
  • Bottom tab bar — persistent navigation for My Pools, Browse, New Pool, and Manage
  • Manage button — direct ⚙️ Manage shortcut on your hosted pool cards in My Pools
  • YOU HOST badge — public pool cards show a YOU HOST indicator when browsing pools you created
  • Help, About, and Release Notes pages
IMPROVEMENTS
  • Public pool browse no longer hides pools you host — your own public pools are visible with a YOU HOST badge
  • Leaderboard entries now show Display Name, falling back to first/last name, then username
  • Host Dashboard payout report uses Display Name consistently
  • Pool cards in Browse show host's display name instead of raw username
  • Tab bar adds padding below all main screens so content is never hidden behind navigation
BUG FIXES
  • Fixed avatar not displaying in header pill (showed default emoji instead of user's selected avatar)
  • Fixed profile fields appearing blank when display name or name was already set
v1.2 JUNE 2026 IMPROVEMENT
NEW FEATURES
  • Pick Entry screen — browse tournament field, search golfers by name, select 10 picks
  • Live leaderboard with earnings, rank, and pick status for each entry
  • Host Dashboard — Payments tab to mark entries paid/unpaid, Payout Report tab
  • Pool locking — host can lock picks before the tournament starts
  • Public pool browse — see open pools accepting entries
  • Pool detail screen — view all entries, their picks, and current standings
IMPROVEMENTS
  • Pull-to-refresh on leaderboard for live tournament updates
  • Golfer field sorted by ESPN world ranking within each tier
  • Ties handled correctly — tied golfers split combined earnings per PGA Tour rules
v1.1 MAY 2026 FIX
BUG FIXES
  • Fixed ESPN API field not loading when tournament started but hadn't completed a round
  • Pool join code now case-insensitive
  • Fixed payout percentages not summing correctly in edge cases
  • Shared auth with GameGrid/GridIron — single account for all YourGameSpot apps
v1.0 APRIL 2026 LAUNCH
INITIAL RELEASE
  • Create PGA Tour golf pick'em pools for any upcoming tournament
  • Players select 10 golfers; prize earnings determine pool standings
  • ESPN Golf API integration for live field and tournament results
  • Standard PGA Tour payout table (1st = 18%, 2nd = 10.9%, …)
  • Public and private pools with 6-character join codes
  • Configurable entry fees, entries per player, and payout splits
  • Shared YourGameSpot account — same login as GameGrid, GridIron, and GameNight
